# Amazon Laptop Market Analysis via Web Scraping and EDA

This project uses Python to scrape laptop listings from Amazon India, clean the data, and perform exploratory data analysis (EDA) to uncover pricing trends and feature correlations.

## Objectives
- Scrape laptop data using BeautifulSoup and requests
- Clean and structure the dataset using pandas and regex
- Analyze features like brand, RAM, ROM, processor, display size, price, and rating
- Visualize insights using seaborn and matplotlib

## Key Insights
- DELL, HP, and LENOVO dominate the listings
- Higher RAM and SSD storage correlate with higher prices
- Windows 11 is the most common OS
- Display sizes cluster around 15.6 inches

## Tools Used
- Python
- BeautifulSoup
- requests
- pandas
- seaborn
- matplotlib

## Files Included
- `Amazon_Laptop_EDA.ipynb`: Full scraping and analysis notebook
- `Cleaned_AmazonData_For_Visualization.csv`: Final dataset
- `requirements.txt`: Python libraries used
